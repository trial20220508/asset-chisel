# シナリオ管理機能 - ユーザーガイド

Asset Chiselのシナリオ管理機能により、複数の資産計画を保存・管理できます。

## 📋 機能概要

### 自動保存
- データを入力すると**1秒後に自動的に保存**されます
- ページを閉じても、次回開いたときにデータが復元されます
- 同じブラウザであれば、データは永続的に保存されます

### 複数シナリオ管理
- 異なる資産計画を複数作成できます
- シナリオごとに名前をつけて管理
- 例：「マイホーム購入プラン」「老後資金プラン」「子供の教育資金」

### エクスポート/インポート
- シナリオをJSONファイルとしてダウンロード
- 別の端末で使用したい場合、ファイルをアップロード
- バックアップとしても活用可能

## 🎯 基本的な使い方

### 1. シナリオの切り替え

画面上部の**シナリオ選択ドロップダウン**から切り替えます。

```
💾 シナリオ: [マイホーム購入 ▼]
```

### 2. シナリオ名の変更

1. シナリオ名の右にある **✏️** アイコンをクリック
2. 新しい名前を入力
3. 「保存」ボタンまたはEnterキーで確定

### 3. 新しいシナリオを作成

**[新規]** ボタンをクリック
- 初期状態のシナリオが作成されます
- すぐに編集を開始できます

### 4. シナリオを保存

**[保存]** ボタンをクリック
- 現在の状態がlocalStorageに保存されます
- （自動保存も有効なので、通常は手動保存不要）

### 5. シナリオをエクスポート

**[出力]** ボタンをクリック
- JSONファイルがダウンロードされます
- ファイル名: `シナリオ名_2026-01-09.json`

### 6. シナリオをインポート

**[読込]** ボタンをクリック
1. ファイル選択ダイアログが表示
2. 以前エクスポートしたJSONファイルを選択
3. 新しいシナリオとして読み込まれます

### 7. シナリオを削除

**[削除]** ボタンをクリック
- 確認ダイアログが表示されます
- 削除すると復元できないので注意
- （最後の1つは削除できません）

## 💡 活用例

### 例1: 複数の人生プランを比較

```
シナリオ1: 「30歳で結婚・マイホーム購入」
- NISA: 月5万円を30年間
- イベント: 5年後に頭金500万円

シナリオ2: 「35歳まで独身・投資重視」
- NISA: 月10万円を25年間
- 個別株: 月3万円
```

切り替えて比較することで、どちらが有利か確認できます。

### 例2: 悲観的シナリオと楽観的シナリオ

```
シナリオA: 「保守的（年利3%）」
シナリオB: 「楽観的（年利7%）」
```

リスクを考慮した複数パターンを作成

### 例3: 家族間での共有

1. PCでシナリオを作成
2. エクスポートしてファイルをスマホに送信
3. スマホでインポート
4. 外出先でもチェック可能

## 📱 端末間でのデータ移行

### 方法1: JSONファイルで移行

**PCからスマホへ：**
1. PCで [出力] ボタンをクリック
2. JSONファイルをダウンロード
3. ファイルをスマホに送信（メール、LINE、Googleドライブなど）
4. スマホで Asset Chisel を開く
5. [読込] ボタンでファイルを選択

**スマホからPCへ：**
同じ手順で逆方向に移行

### 方法2: クラウドストレージ経由

1. PCでエクスポート
2. Googleドライブ/Dropboxにアップロード
3. スマホでダウンロード
4. インポート

## 🔒 データの保存場所

### localStorageに保存
- データはブラウザのlocalStorageに保存されます
- 同じブラウザ・同じデバイスでのみ有効
- ブラウザのデータを削除すると消えます

### プライバシー
- データはあなたのデバイスにのみ保存
- サーバーには送信されません
- 完全にオフラインで動作

---

## ⚠️ 注意事項

### ブラウザのデータ削除
以下の操作でデータが消えます：
- ブラウザの履歴・キャッシュをクリア
- プライベートモード/シークレットモードを使用
- ブラウザのリセット

**対策：** 定期的にエクスポートしてバックアップ

### ブラウザ間での共有
- Chrome、Safari、Firefoxなど、ブラウザが異なるとデータは共有されません
- 異なるブラウザで使う場合は、エクスポート/インポートで移行

### ストレージ容量
- localStorageの容量は通常5〜10MB
- 大量のシナリオを作ると容量不足になる可能性
- 不要なシナリオは削除推奨

## 🆘 トラブルシューティング

### Q: データが消えた
**A:** ブラウザのキャッシュクリアが原因かもしれません
- 以前エクスポートしたJSONファイルがあれば、インポートで復元
- 今後は定期的にバックアップを取ることを推奨

### Q: 別の端末で使いたい
**A:** エクスポート/インポート機能を使用
1. 現在の端末で [出力] をクリック
2. ファイルを別端末に転送
3. 別端末で [読込] をクリック

### Q: シナリオが多すぎて管理が大変
**A:** 不要なシナリオは削除しましょう
- 削除前にエクスポートしてアーカイブ
- 必要になったら再度インポート

### Q: 自動保存を無効にしたい
**A:** 現在、自動保存は常に有効です
- 手動保存も併用できます
- 将来のアップデートで設定可能にする予定
